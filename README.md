# 🚑 AmbuPass

## 📌 Problem
In emergency situations, people face delays in getting help due to lack of quick communication and hospital availability information.

## 💡 Solution
AmbuPass is a smart emergency response system that instantly alerts nearby hospitals and provides real-time assistance.

## 🚀 Features
- ⚡ Instant emergency alert system
- 📍 Location sharing
- 🏥 Hospital connection
- 👤 User-friendly interface
